Crude Oil Production & GDP Impact on Global Warming

Description:

The project’s primary purpose is to identify if factors such as crude oil production and GDP growth impact the global temperature, contributing to the melting of Arctic and Antarctic sea ice. Using real-world data, this team developed multiple hypotheses regarding the impact on global temperatures and plotted data points to find correlations between crude oil production and GDP growth.

  • Data cleaning
  • Data analysis
  • Data visualization
  • Predictive analysis

  • Pandas – Data cleaning
  • NumPy – Calculations (standard deviation, mean values, random number generator)
  • Plotly – Plot visualization
  • Mapbox – Map API used to plot maps with Plotly
  • Altair – Correlation graphs

Tokenization of Private Equity

Description:

The team created a platform that allows any investor to participate in private equity funds (not easily converted into cash) and offer a selection of recommendations. They simulated expected returns using Autoregressive Moving Average (ARMA) and Autoregressive Integrated Moving Average (ARIMA) time series models and long short-term memory (LSTM) recurrent neural network (RNN) neural models. They also deployed a decentralized app to host a dashboard.

  • Data preparation
  • Data analysis
  • Modelling with ARMA, ARIMA, and LSTM
  • Decentralized application

  • Remix IDE (Solidity)
  • OpenZepellin
  • Python
  • Infura
  • Blockchain – Kovan Testnet
  • MetaMask

Nasdaq Predictive Model

Description:

The project objective was to build something to analyze live data and send a daily list of stock tickers predicted to increase in value the following day. The team wanted to identify stocks expected to outperform the Nasdaq, then create a way to send alerts with the ticker symbols of those stocks.
